# Resume Analyser

An AI-powered resume analysis tool that reviews resumes, 
provides section-wise critique, and suggests improvements.

## Features
- Upload and parse resume content
- Section-wise scoring and feedback
- Suggested improvements for each section
- Career tips and interview question suggestions

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS

## How to Run
1. Clone this repository
2. Run `npm install`
3. Run `npm run dev`
